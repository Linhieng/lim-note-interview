# 必刷算法

## 必背常识

除以二，向下取整（偏左⇦）：
- `leftIndex + rightIndex >>> 1`

除以二，向上取整（偏右⇨）：
- `leftIndex + rightIndex + 1 >>> 1`

向下取整：
- `~~(1.1)`
- `~~(1.9)`

ASCII 数值：
- 0-9 是 48 - 57  / 0x30 - 0x39
- A-Z 是 65 - 90  / 0x41 - 0x5A
- a-z 是 97 - 122 / 0x61 - 0x7A
