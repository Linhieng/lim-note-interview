## [03 无重复字符的最长子串](https://leetcode.cn/problems/longest-substring-without-repeating-characters/)

### Python3

【官方题解滑动窗口】：
```py
class Solution:
    def lengthOfLongestSubstring(self, s: str) -> int:
        n = len(s)
        if n < 2:
            return n

        occur = set() # 记录当前子串内的字符
        right, max_len = 0, 1

        for left in range(n):
            if left != 0:
                occur.remove(s[left - 1])

            while right < n and s[right] not in occur:
                occur.add(s[right])
                right += 1

            # [left, right) 是当前不含重复字符的子串
            max_len = max(max_len, right - left)

            if right == n:
                return max_len

        return max_len
```

【我的滑动窗口】：
```py
class Solution:
    def update(self, str, char):
        for i, c in enumerate(str):
            if c == char:
                return str[i+1:] + char
        return str + char

    def lengthOfLongestSubstring(self, s: str) -> int:
        str = ''
        res = 0
        for char in s:
            str = self.update(str, char)
            res = max(res, len(str))
        return res
```
